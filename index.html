AI Setup

API keys are stored securely in .env on the server — never exposed to the browser

The selected provider is used for Hook, Title, Description, Hashtags & Reimagine. Your choice is saved automatically.

⚡ Grok AI (xAI) — Premium text rewriting:
  1. Go to console.x.ai and sign in with your X (Twitter) account
  2. Click "API Keys" → "Create API Key"
  3. Open .env in the project folder and set:
    GROK_API_KEY=xai-…your-key…
  4. Restart the server (python server.py)

⭐ Uses grok-3-mini — fast & cost-efficient. Requires X Premium subscription.

📌 Gemini AI (Google) — Free text rewriting:
  1. Go to aistudio.google.com/app/apikey
  2. Sign in with Google (free) → click "Create API Key"
  3. Open .env in the project folder and set:
    GEMINI_API_KEY=AIzaSy…your-key…
  4. Restart the server (python server.py)

🎨 Remove.bg — Free background removal:
  1. Go to remove.bg/api
  2. Sign up free → go to API Keys → copy your key
  3. Open .env and set:
    REMOVEBG_API_KEY=abc123…your-key…
  4. Restart the server (python server.py)

✅ Free: 50 background removals/month · No credit card needed

🔒 Security note:

Keys are stored only in .env on the server. They are never sent to the browser, never visible in DevTools, and never stored in localStorage. Make sure .env is listed in .gitignore so it is never committed to Git.
